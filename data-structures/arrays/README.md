# Arrays

Practice problems and implementations for array data structure.

## Topics Covered
- Basic operations (insertion, deletion)
- Searching (linear search, binary search)
- Two Pointers technique
- Sliding Window
- Array manipulation

## Files
| File | Description |
|------|-------------|
| `array_duplication.c` | Remove duplicates from array |
| `deletion.c` | Delete elements |
| `insertion.c` | Insert elements |
| `linearsearch.c` | Linear search |
| `linear_search_recursion.c` | Recursive linear search |
| `max_min.c` | Find max/min |
| `pair_sum.c` | Pair sum problems |
| `pairsumusing_hash.c` | Hash-based pair sum |
| `reverse_inplace.c` | In-place reversal |
